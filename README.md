## FAQ
- v1.10.10-lts之前的版本可以直接迁移到v2么？  
答：不可以，必须升级到v1.10.10-lts之后再迁移。
- 可以直接迁移到v2.2.0版本么?  
答：不可以，必须先迁移到v2.1.0，再升级到v2.2.0。
- 社区版可以迁移到专业版么？  
答：不可以，v1社区版只能迁移到v2社区版，v1专业版只能迁移到v2专业版，且v1和v2的专业版必须有有效的license许可。
